# API Testing Project - Simple Books API

## 📌 Objective
To perform API testing using Postman for the Simple Books API.

## 🔧 Tools Used
- Postman
- Newman (optional)
- Excel (for test cases)

## 📂 APIs Tested
- GET /status
- GET /books
- GET /books/{id}
- POST /api-clients
- POST /orders
- GET /orders
- PATCH /orders/{id}
- DELETE /orders/{id}

## 🧪 Testing Performed
- Status code validation
- Response validation
- CRUD operations testing
- Authorization testing
- Negative testing

## 📊 Test Deliverables
- Postman Collection
- API Test Cases
- Screenshots

## 🚀 Execution
APIs were tested using Postman Collection Runner with automated test scripts.

## 📌 Conclusion
All APIs were tested successfully with both positive and negative scenarios.
